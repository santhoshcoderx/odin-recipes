# Odin Recipes
A simple multi-page website built as part of The Odin Project curriculum. This project focuses on practicing basic HTML structure, navigation between pages, and semantic markup.

## About
Odin Recipes is a small recipe collection site. The homepage lists links to individual recipe pages, each containing an image, ingredients, and step-by-step instructions.

## Pages
- Home (index.html) — Lists all available recipes
- Slow Cooked Barbecue Chicken Breast (recipes/barbecue.html)
- Baked BBQ Chicken Drumsticks (recipes/baked_chicken.html)
- Easy Oven-Barbecued Beef Brisket (recipes/beef_brisket.html)

## What I Practiced
- Basic HTML document structure
- Linking between multiple HTML pages
- Using semantic tags (h1, ul, li, a, img)
- Organizing files into folders
- Writing descriptive alt text for images
- Using Git and GitHub for version control

## Tech Stack
- HTML5
- Git / GitHub for version control
